# 拾物 —— 大学校园闲置物品交易小程序

## 产品概述

### 产品定位

「拾物」是一款专为在校大学生设计的校园闲置物品交易微信小程序。聚焦于解决大学生闲置物品处置麻烦、跨校区邮寄成本高等痛点，以"同校面交、安全可信"为核心价值主张，打造一个轻量、高效、可信的校园二手交易社区。

### 核心痛点

| 痛点 | 描述 |
|------|------|
| 闲置物品积压 | 教材、电子产品、生活用品大量闲置，丢弃可惜，变现无门 |
| 邮寄成本高 | 通过闲鱼等平台交易需要邮寄，运费往往超过物品价值 |
| 陌生人信任度低 | 与校外陌生人交易存在安全顾虑 |
| 交易效率低 | 现有平台覆盖范围广，难以找到同校买家 |

### 目标用户

- **主要用户**：在校大学生，年龄 18-24 岁
- **用户特征**：
  - 有大量闲置教材、电子产品、生活用品
  - 熟悉微信生态，习惯移动端操作
  - 价格敏感，追求性价比
  - 更愿意与同学/校友交易，信任度更高
  - 倾向面交，避免邮寄麻烦

### 核心功能

1. **发布商品**：拍照上传、商品描述、自定义定价、选择校区/地点
2. **分类浏览/搜索**：按分类、价格、校区等多维度筛选商品
3. **私信洽谈 + 预约面交**：内置 IM 私信，支持约定面交时间和地点
4. **校区/地点筛选**：精准匹配同校区买卖双方，减少无效沟通
5. **诚信评分系统**：交易完成后双向评价，累积信用分，提升交易信任度

### 产品价值主张

- **卖家**：快速变现闲置，0 邮寄成本，安全面交
- **买家**：低价淘好物，同学来源更可信，面交验货更放心
- **平台**：校园闭环生态，高频刚需场景，用户粘性强

### 商业模式

- **当前阶段（0-12 个月）**：完全免费，积累用户和口碑
- **商业化阶段（12 个月后）**：
  - 成交额 **2% 服务费**（买家或卖家支付，待定）
  - 后期可探索：置顶推广、校园广告、增值会员服务

### 平台信息

- **载体**：微信小程序
- **依托生态**：微信登录（学号实名认证）、微信支付、微信 IM
- **目标覆盖**：全国高校（优先深耕重点城市高校）

### 竞品分析

| 维度 | 拾物 | 闲鱼 | 二手玫瑰（校园版） |
|------|------|------|-----------------|
| 目标用户 | 在校大学生 | 全人群 | 大学生 |
| 地域范围 | 校园内/校区间 | 全国 | 全国高校 |
| 面交支持 | 强（核心功能） | 弱 | 中 |
| 信任机制 | 学号实名+评分 | 芝麻信用 | 校园认证 |
| 邮寄需求 | 低（面交优先） | 高 | 中 |
| 微信生态 | 原生小程序 | 独立 App | 小程序 |

### 成功指标（北极星）

- **核心指标**：月活跃用户数（MAU）
- **次要指标**：
  - 发布商品数 / 月
  - 成交率（发布商品中完成交易的比例）
  - 用户留存率（30 日留存）
  - 平均交易完成时长
