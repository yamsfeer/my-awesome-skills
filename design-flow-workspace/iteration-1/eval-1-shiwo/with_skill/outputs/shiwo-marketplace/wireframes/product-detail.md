## 约束汇总

| 标记 | 约束 |
|------|------|
| 🔴 | 图片轮播全宽 1:1 比例 |
| 🔴 | 价格大字主色，不可用灰色 |
| 🔴 | 商品标题完整显示，不截断 |
| 🔴 | 已售出/已预约商品图片必须加蒙层 + 状态文字 |
| 🔴 | 底部操作栏固定，不被内容遮挡 |
| 🔴 | 卖家本人不显示"联系卖家"按钮 |
| 🔴 | 下架操作必须二次确认弹窗 |
| 🔵 | 图片数量 > 1 时显示小圆点指示器 |
| 🔵 | 描述超 5 行显示"展开"按钮 |
| 🔵 | 点击卖家信息跳转其个人主页 |
| 🟢 | 显示原价（有删除线）引导折扣感知 |
| 🟢 | 推荐相似商品横向滚动列表 |
| 🟢 | 浏览量展示增强商品可信度 |
