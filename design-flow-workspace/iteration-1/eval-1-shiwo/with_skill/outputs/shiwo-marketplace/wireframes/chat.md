## 约束汇总

| 标记 | 约束 |
|------|------|
| 🔴 | 商品信息条固定在顶部，始终可见 |
| 🔴 | 输入栏固定在底部，键盘弹出时随键盘上移 |
| 🔴 | 预约面交按钮仅商品在售时显示 |
| 🔴 | 预约卡片必须显示时间/地点/状态 |
| 🔴 | 发送失败消息显示红色感叹号，支持重发 |
| 🔴 | 预约面交时间和地点均为必填项 |
| 🔵 | 自己消息在右侧（主色背景），对方在左侧（白色背景） |
| 🔵 | 消息间隔 > 5 分钟显示时间戳 |
| 🔵 | 进入页面自动滚动到最新消息 |
| 🟢 | 自己发送的消息显示已读/未读状态 |
| 🟢 | 快捷地点选择（图书馆/食堂/宿舍楼等校园常用地点） |
| 🟢 | 上拉加载更早的历史消息 |
