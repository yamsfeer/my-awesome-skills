## 约束汇总

| 标记 | 约束 |
|------|------|
| 🔴 | 搜索栏固定在顶部，滚动时不消失 |
| 🔴 | 成色角标必须显示在图片左上角 |
| 🔴 | 已售出/已预约商品必须显示半透明蒙层 |
| 🔴 | 价格用主色高亮显示（不可用灰色） |
| 🔴 | 底部 TabBar 固定，不可被内容遮挡 |
| 🔵 | 双列瀑布流，图片比例约 3:4 |
| 🔵 | 标题最多 2 行，超出省略号 |
| 🔵 | 分类 Tab 横向滚动，同时显示 4-5 个 |
| 🟢 | 图片懒加载，先显示灰色占位 |
| 🟢 | 空状态建议加插画增强视觉引导 |
| 🟢 | 上拉加载更多，避免分页按钮 |
