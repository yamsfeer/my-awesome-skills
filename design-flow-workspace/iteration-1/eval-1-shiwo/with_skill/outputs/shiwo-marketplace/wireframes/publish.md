## 约束汇总

| 标记 | 约束 |
|------|------|
| 🔴 | 图片至少 1 张，最多 9 张（超出时隐藏"+"按钮） |
| 🔴 | 标题、分类、成色、价格为必填项 |
| 🔴 | 发布/保存草稿前必须完成必填项校验 |
| 🔴 | 校区信息只读，从用户资料自动获取 |
| 🔴 | 取消按钮必须存在，防止误操作 |
| 🔵 | 价格字段弹出数字键盘 |
| 🔵 | 标题实时显示字数（x/60） |
| 🔵 | 描述实时显示字数（x/1000） |
| 🔵 | 错误字段边框变红，错误提示在字段下方 |
| 🟢 | 原价字段选填，填写后自动展示折扣率 |
| 🟢 | 图片支持拖拽排序 |
| 🟢 | 提供描述模板快速填写 |
