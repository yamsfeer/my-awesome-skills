## 约束说明

| 标记 | 约束内容 |
|------|---------|
| 🔴 | 个人中心仅本人可访问（登录态路由守卫） |
| 🔴 | 手机号必须脱敏显示（138****8888格式） |
| 🔵 | 基础版用户顶部始终展示升级引导 Banner |
| 🔵 | 体脂数据折线仅高级版用户可见 |
| 🔵 | 成就网格：已获得彩色展示，未获得灰色半透明 + 锁图标 |
| 🔵 | 体重趋势图至少需要 2 条记录才展示（否则显示引导录入） |
| 🟢 | 高级版用户"到期日期"距到期 7 天内变红色提醒 |
| 🟢 | 数据卡片点击可查看详细历史记录 |
