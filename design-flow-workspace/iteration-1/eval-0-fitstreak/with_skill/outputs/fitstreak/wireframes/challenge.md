## 约束说明

| 标记 | 约束内容 |
|------|---------|
| 🔴 | 挑战时长固定选项：7天、21天、30天 |
| 🔴 | 邀请好友数量：至少 1 人，最多 9 人（含自己共 10 人） |
| 🔴 | 进行中挑战不可退出，待开始挑战可退出 |
| 🔴 | 基础版好友上限 5 人，发起挑战时校验 |
| 🔵 | 挑战排行榜按当前连续打卡天数排序 |
| 🔵 | 挑战进度条 = 已过天数 / 总天数 |
| 🟢 | 挑战结束后可查看历史排行榜 |
| 🟢 | 建议挑战结束时推送通知给所有参与者 |
