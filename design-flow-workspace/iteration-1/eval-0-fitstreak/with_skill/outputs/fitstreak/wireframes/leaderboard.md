## 约束说明

| 标记 | 约束内容 |
|------|---------|
| 🔴 | 排行榜仅展示好友（含自己），不做公开全站榜 |
| 🔴 | 未登录用户不可访问此页面，跳转登录页 |
| 🔵 | Tab 切换：本周（自然周）/ 本月 / 总榜（全部时间连续天数） |
| 🔵 | Top 3 领奖台：第一名居中，高度最高，金色头像边框 |
| 🔵 | 自己若在 Top 3 内，头像添加"你"标签 |
| 🔵 | 自己的排名条目若不在可见区域，底部固定显示自己排名 |
| 🟢 | 排行数据建议每 5 分钟刷新一次（非实时）|
| 🟢 | 点击其他用户头像可跳转其公开主页 |
