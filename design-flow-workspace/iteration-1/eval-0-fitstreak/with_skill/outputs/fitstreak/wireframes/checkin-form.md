## 约束说明

| 标记 | 约束内容 |
|------|---------|
| 🔴 | 运动类型为必填，未选择时提交按钮置灰不可点击 |
| 🔴 | 提交进行中禁止重复点击、禁止返回（防止重复打卡） |
| 🔴 | 同一用户同一天只能打卡一次（由后端 409 错误拦截） |
| 🔴 | 打卡日期必须为今天或昨天（不允许补打更早日期） |
| 🔵 | 时长滑块范围 5-480 分钟，低于 5 分钟不计有效打卡 |
| 🔵 | 图片上传大小限制 10MB，客户端上传前校验 |
| 🔵 | 备注最多 200 字，超出时输入被截断 |
| 🟢 | 成功动效持续约 2 秒后自动跳转首页 |
| 🟢 | 如有新成就解锁，成功页面展示成就卡片 |
