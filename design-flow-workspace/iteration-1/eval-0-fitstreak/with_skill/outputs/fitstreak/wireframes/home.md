## 约束说明

| 标记 | 约束内容 |
|------|---------|
| 🔴 | 底部导航栏固定，不随页面滚动 |
| 🔴 | 今日打卡按钮：已打卡时变为绿色不可点击状态 |
| 🔴 | 月历仅展示当前月及前后月，不可超出范围 |
| 🔵 | 连续天数为 0 时，数字区域显示"开始你的第一天"文案 |
| 🔵 | 最近记录列表最多显示 5 条，"查看全部"跳转打卡历史页 |
| 🟢 | 激励文案建议每次随机抽取，避免重复 |
| 🟢 | 已打卡状态建议配合微动效（如绿色渐入、数字增加动画）提升情绪价值 |
