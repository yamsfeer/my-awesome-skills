# FitStreak - 产品概述

## 产品定位

FitStreak 是一款面向上班族的移动健身打卡应用，通过社交激励、成就可视化和好友挑战，帮助用户建立并坚持每日运动习惯。

## 核心问题

健身爱好者难以坚持运动计划，缺乏社交激励和进度可视化——知道要健身，却总是三天打鱼两天晒网。

## 目标用户

| 维度 | 描述 |
|------|------|
| 年龄 | 18-35 岁 |
| 职业 | 上班族（写字楼、互联网从业者为主） |
| 健身意愿 | 有意愿但自律性不足，需要外部激励 |
| 运动时间 | 每天 30-60 分钟（晨跑、健身房、居家锻炼） |
| 痛点 | 坚持不下去、看不到进步、缺乏同伴监督 |

**典型用户画像：**
小李，28 岁，互联网公司产品经理，工作日晚上在附近健身房锻炼 40 分钟，但时常因加班或懒惰中断，希望有人"盯着"自己打卡，也想看到自己的体型变化趋势。

## 平台

iOS + Android（React Native 跨端开发）

## 商业模式

| 版本 | 价格 | 说明 |
|------|------|------|
| 基础版 | 免费 | 每日打卡、基础统计、最多 5 位好友 |
| 高级版 | ¥18/月 | 解锁高级统计图表、无限好友、专属成就徽章、AI 计划推荐 |

订阅制：按月订阅，支持微信支付 / 支付宝 / App Store 内购。

## 核心功能模块

1. **每日打卡记录**：一键打卡当天运动，记录运动类型（跑步/健身/游泳等）、时长、心率；支持图片上传
2. **运动计划制定**：根据目标（减脂/增肌/维持）生成周计划，支持自定义修改；高级版支持 AI 智能推荐
3. **好友挑战 / 排行榜**：发起好友连续打卡挑战（7天/21天/30天），实时排行榜展示连续打卡天数
4. **成就系统 / 徽章**：打卡里程碑徽章（坚持 7 天、30 天等）、特殊成就（燃烧战士、铁人等），社交分享
5. **体重 / 体脂趋势图**：录入身体数据，可视化折线图展示变化趋势，高级版支持更精细的数据分析
