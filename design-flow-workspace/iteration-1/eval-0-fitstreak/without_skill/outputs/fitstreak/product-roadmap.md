## Phase 1：MVP 开发（2026 Q2，约 12 周）

**目标：** 完成核心功能开发，通过内测验证产品价值

### Sprint 1-2（Week 1-4）：基础架构 + 用户系统
- [ ] 技术框架搭建（React Native + Node.js 后端）
- [ ] 用户注册/登录（手机号 + Apple/Google 一键登录）
- [ ] 用户 Profile 页面（头像、昵称、目标设定）
- [ ] CI/CD 流水线搭建

### Sprint 3-4（Week 5-8）：核心打卡功能
- [ ] 运动类型库（20+ 种运动）
- [ ] 每日打卡流程（选择运动 → 记录时长 → 完成打卡）
- [ ] Streak 连续天数计算逻辑
- [ ] 打卡日历热力图（首页核心组件）
- [ ] 基础成就系统（7/30/100 天徽章）

### Sprint 5-6（Week 9-12）：社交 + 内测
- [ ] 好友关系（添加/删除/查看动态）
- [ ] 基础排行榜（本周打卡次数）
- [ ] Push 通知（每日提醒）
- [ ] 内测版本发布（TestFlight + 内部测试群）
- [ ] 用户反馈收集与 Bug 修复

**交付物：** 内测 APK + TestFlight 链接

## Phase 2：公测上线（2026 Q3，约 8 周）

**目标：** 上架应用商店，获取前 10,000 DAU，验证付费转化

### Week 1-3：功能完善
- [ ] 运动计划模板库（10套免费 + 5套 Premium）
- [ ] 体重/体脂数据录入与趋势图（Premium）
- [ ] 好友挑战赛功能（Premium）
- [ ] 订阅购买流程（Apple Pay / 微信支付 / 支付宝）
- [ ] 引导页 Onboarding（3步完成目标设定）

### Week 4-5：App Store 准备
- [ ] App Store / Google Play 应用审核材料准备
- [ ] ASO 优化（关键词：健身打卡、运动计划、健康习惯）
- [ ] 隐私政策 + 用户协议
- [ ] 应用截图 + 宣传视频制作

### Week 6-8：上线 + 冷启动运营
- [ ] 正式上线 iOS + Android
- [ ] 小红书/抖音种子用户获取活动
- [ ] 邀请好友送 Premium 7 天体验活动
- [ ] 数据监控看板搭建

**关键指标：**
- 下载量：5,000+
- DAU：2,000+
- D7 留存：40%+
- 付费转化：5%+

## Phase 3：增长优化（2026 Q4，约 12 周）

**目标：** 冲击 DAU 20,000，优化留存，提升付费率至 8%

### 功能迭代
- [ ] **打卡补签机制**：Premium 用户每月可补签 3 次（防止 Streak 断裂流失）
- [ ] **社群挑战**：官方发起全国打卡挑战赛（如"百日健身挑战"）
- [ ] **分享卡片**：生成精美打卡海报，一键分享朋友圈/微博
- [ ] **运动提醒升级**：智能时间推荐（基于用户历史打卡时间）
- [ ] **周/月总结报告**：图文并茂的数据报告（Premium）
- [ ] **运动日记**：打卡时可附文字/图片记录

### 增长实验
- [ ] A/B 测试：Onboarding 流程优化（目标 D1 留存 +10%）
- [ ] A/B 测试：Premium 订阅弹窗时机优化
- [ ] Referral 裂变：邀请好友双方各得 14 天 Premium
- [ ] Push 通知个性化（基于用户行为触发）

**关键指标：**
- DAU：20,000+
- D30 留存：35%+
- 付费转化：8%+
- 月收入：¥50,000+

## Phase 4：高级功能（2027 Q1，约 10 周）

**目标：** 提升产品壁垒，扩大 Premium 价值

### 功能方向
- [ ] **AI 智能计划生成**：输入目标（减脂/增肌/塑形）+ 可用时间，AI 生成个性化周计划
- [ ] **Apple Watch / Wear OS 支持**：表盘打卡、心率数据同步
- [ ] **HealthKit / Google Fit 集成**：自动同步步数、运动数据
- [ ] **打卡地图**：标记运动地点，形成个人运动足迹
- [ ] **语音播报**：运动计时结束语音提示
- [ ] **家庭版订阅**：支持最多 6 个账号共享 Premium

### 技术优化
- [ ] 离线模式：无网络时可打卡，联网后同步
- [ ] 性能优化：首屏加载 < 2s，Crash Rate < 0.1%

**关键指标：**
- DAU：40,000+
- Watch 版用户占比：15%+
- 家庭版订阅：500+ 组

## Phase 5：商业化深化（2027 Q2，约 12 周）

**目标：** 月收入突破 ¥200,000，探索 B 端市场

### 功能方向
- [ ] **企业健康挑战版**：为企业提供员工健身打卡管理后台
- [ ] **品牌联名徽章**：与运动品牌合作（运动鞋/健身房）发放联名成就
- [ ] **直播打卡房间**：好友可约在线同时打卡
- [ ] **课程商城**：引入第三方健身课程，抽佣模式
- [ ] **数据开放 API**：为健身设备厂商提供数据接入

### 商业化实验
- [ ] B 端企业版 POC（3家种子企业合作）
- [ ] 广告系统精细化（Premium 免广告价值提升）
