# 打卡流程线框图（Checkin Screen Wireframe）

> 版本：v1.0 | 平台：iOS + Android | 更新：2026-03-22

---

## 页面基本信息

| 属性 | 值 |
|------|-----|
| 触发方式 | 首页"立即打卡"按钮 |
| 展示形式 | Bottom Sheet Modal（底部弹出） |
| 步骤数 | 3步（运动选择 → 时长 → 备注） |
| 关闭方式 | 向下滑动 / 点击遮罩 |

---

## 整体流程概览

```
首页卡片
  [立即打卡]
      │
      ▼
Step 1: 选择运动类型
      │ 选择运动
      ▼
Step 2: 输入运动时长
      │ 确认时长
      ▼
Step 3: 添加备注（可选）
      │ 完成打卡 / 跳过
      ▼
 打卡成功弹窗
      │ 关闭 / 分享
      ▼
 返回首页（数据刷新）
```

- 计时器按钮：次级按钮样式，点击后切换到计时器模式
- "下一步"按钮：选中时长后激活，默认禁用

## 键盘处理

- 当备注输入框获取焦点时，整个 Bottom Sheet 向上移动，避免键盘遮挡
- 使用 `KeyboardAvoidingView` 实现
- 点击非输入区域收起键盘
