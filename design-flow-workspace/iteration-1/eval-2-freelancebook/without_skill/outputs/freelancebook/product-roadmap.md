## Phase 1：MVP（2026-05，公开上线）

**目标**：交付核心闭环，支撑早期用户验证。

### 功能清单

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 认证 | 注册 / 登录 / 找回密码 / 30 天试用激活 | P0 |
| 客户管理 | 创建、编辑、归档客户；多联系人管理 | P0 |
| 发票管理 | 创建发票、状态流转（draft→sent→paid/overdue） | P0 |
| 收款记录 | 登记实收款项、关联发票 | P0 |
| 收入统计 | 月度收入汇总卡片、收款状态汇总 | P0 |
| 订阅计费 | 订阅购买、续费、降级到只读 | P0 |
| 税务预估 | 简版个税速算（年度收入 × 税率表） | P1 |
| 通知 | 逾期发票站内提醒 | P1 |

### MVP 验收标准
- 用户可完成「新建客户 → 创建发票 → 标记已收款」全流程，耗时 < 3 分钟
- 首页加载 LCP < 2.5s（桌面网络）
- 数据导出（CSV）可用

---

## Phase 2：增长功能（2026 Q3）

**目标**：提升留存与付费转化，扩大用户基数。

| 功能 | 说明 |
|------|------|
| 邀请裂变 | 邀请链接 + 双方延长 15 天试用 |
| 发票 PDF 导出 | 生成标准格式发票 PDF，一键下载 |
| 项目 / 合同模块 | 合同记录、里程碑拆付款 |
| 收入图表增强 | 按客户 / 项目维度饼图、趋势折线图 |
| 数据导入 | 从 Excel/CSV 批量导入历史收款记录 |
| 邮件提醒 | 逾期发票自动发送提醒邮件给客户 |
| 多货币支持（预研） | 港币、美元标注（显示，不做汇率换算） |

---

## Phase 3：税务深化 + 移动端（2026 Q4）

**目标**：打造税务核心差异化，拓展移动使用场景。

| 功能 | 说明 |
|------|------|
| 税务预估增强 | 支持增值税小规模 / 一般纳税人、季度预缴提醒 |
| 年度报税报告 | 一键生成全年收入汇总 PDF，可直接交给会计 |
| 自定义税率模板 | 按省份 / 注册类型保存不同计算方案 |
| PWA / 移动适配 | 手机端快速登记收款，响应式优化 |
| 收据扫描（OCR） | 拍照上传收据，自动识别金额（第三方 OCR） |

---

## Phase 4：团队协作 / 企业版（2027 Q1）

**目标**：向「小型工作室 / 虚拟团队」拓展，提升 ARPU。

| 功能 | 说明 |
|------|------|
| 团队成员管理 | 邀请协作成员，分配只读 / 编辑权限 |
| 多工作空间 | 一个账号管理多个独立业务主体 |
| 会计对接 API | 开放 Webhook，对接用友/金蝶等会计软件 |
| 企业版定价 | ¥99/月（5 席位），¥199/月（无限席位） |
| SSO 登录 | 企业 SAML / OIDC 单点登录 |

---

## 技术债务 & 持续优化（贯穿全程）

- 单元测试覆盖率 ≥ 70%
- 金额计算全链路审计（分 → 元显示转换）
- GDPR / 数据隐私合规（敏感字段加密存储）
- 性能监控告警（P99 接口响应 < 500ms）
- 可访问性（WCAG 2.1 AA 级别）
