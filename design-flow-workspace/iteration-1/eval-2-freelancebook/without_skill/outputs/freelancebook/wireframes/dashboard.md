## 交互说明

| 元素 | 交互行为 |
|------|----------|
| 统计卡片（待收款） | 点击跳转到 `/invoices?status=sent` |
| 统计卡片（逾期未收） | 点击跳转到 `/invoices?status=overdue` |
| 趋势图悬浮 | Tooltip 显示当月具体金额 |
| 发票行 | 点击整行打开右侧抽屉（Invoice Detail Drawer） |
| 发票行 Hover | 显示"查看"/"编辑"操作按钮 |
| 快捷操作按钮 | 点击跳转到对应新建页面 |
| 逾期横幅按钮 | 跳转到逾期发票列表 |

---

## 响应式适配（1024px）

```
当宽度 < 1280px：
- 侧边栏折叠为图标模式（icon-only）
- 统计卡片改为 col-6（2 列排列）

当宽度 < 768px（平板）：
- 侧边栏隐藏，顶部汉堡菜单展开
- 统计卡片 col-12（单列）
- 趋势图占满宽度（col-12）
- 快捷操作收入下方
```
