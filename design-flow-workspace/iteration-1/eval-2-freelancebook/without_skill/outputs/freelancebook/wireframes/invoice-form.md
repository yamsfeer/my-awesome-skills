### 区块二：发票明细

```
┌───────────────────────────────────────────────────────────────┐
│  发票明细                                                      │
│                                                                │
│  服务描述                      数量   单价（元）   小计         │
│  ─────────────────────────────────────────────────────────    │
│  ┌──────────────────────────┐  ┌──┐ ┌──────────┐ ┌────────┐  │
│  │ 如：UI设计服务 - 首页    │  │1 │ │ 15000.00 │ │15000.00│ [x]│
│  └──────────────────────────┘  └──┘ └──────────┘ └────────┘  │
│  ┌──────────────────────────┐  ┌──┐ ┌──────────┐ ┌────────┐  │
│  │ 如：交互原型制作          │  │2 │ │  5000.00 │ │10000.00│ [x]│
│  └──────────────────────────┘  └──┘ └──────────┘ └────────┘  │
│                                                                │
│  [+ 添加明细行]                                                │
│                                                                │
│  ─────────────────────────────────────────────────────────    │
│                                     税前金额：  ¥25,000.00    │
│                                     税额 (0%)：    ¥0.00      │
│                                     ─────────────────────     │
│                                     合计：      ¥25,000.00    │
└───────────────────────────────────────────────────────────────┘
```

**明细行说明：**
- 每行末尾 `[x]` 按钮：删除该行（至少保留1行）
- 小计字段为只读计算值（数量 × 单价）
- 拖拽手柄（左侧）可调整排序

## 底部操作栏（固定在页面底部）

```
┌────────────────────────────────────────────────────────────┐
│  [取消]                       [保存草稿]  [创建并发送]      │
└────────────────────────────────────────────────────────────┘
```

| 按钮 | 行为 |
|------|------|
| 取消 | 弹出确认弹窗「是否放弃当前修改？」，确认后返回列表 |
| 保存草稿 | 提交 POST /invoices，status=draft，跳转到发票列表 |
| 创建并发送 | 提交后立即更新 status=sent，弹出成功提示 |

---

## 表单校验规则

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 客户 | 必选 | "请选择客户" |
| 开票日期 | 必填，合法日期 | "请选择开票日期" |
| 付款截止日 | 不得早于开票日 | "截止日期不得早于开票日期" |
| 明细描述 | 必填，1-500字 | "请填写服务描述" |
| 数量 | 必填，> 0 | "数量必须大于0" |
| 单价 | 必填，≥ 0.01元 | "请填写有效金额" |
| 明细行数 | 至少1行 | "请至少添加一条明细" |
