# FreelanceBook 产品概述

## 一、产品定位

FreelanceBook 是一款面向自由职业者的发票与收入管理 SaaS 工具。
它帮助独立开发者、设计师、咨询顾问等个人从业者，统一管理多客户收款记录、生成发票、追踪付款状态，并在年底一键生成税务报告，彻底告别手动记账的混乱。

**核心价值主张：**
> "把账管好，让你专注创作。"

---

## 二、目标用户

| 维度 | 描述 |
|------|------|
| 职业类型 | 独立开发者、UI/UX 设计师、管理咨询顾问、自媒体创作者 |
| 年龄段 | 25 – 45 岁 |
| 月收入区间 | ¥20,000 – ¥50,000 |
| 使用场景 | 多客户并行、收款周期不固定、需要自行报税 |

### 用户痛点

1. **收入来源零散**：同时服务 3–10 个客户，每笔款项来源、金额、周期各不相同，极易遗漏。
2. **发票状态混乱**：邮件里翻来翻去找发票，不知道哪张已收款、哪张已逾期。
3. **年底税务头疼**：汇总全年收入耗时数日，预估应缴税额全靠猜。
4. **工具割裂**：收入记录在 Excel，发票用邮件，合同在文件夹，三处分散。

## 三、核心功能

### 1. 客户管理
- 新建/编辑客户档案（公司名、税号、地址、多联系人）
- 查看单客户历史合同与收款总览
- 支持客户标签分类

### 2. 项目 / 合同记录
- 记录项目名、合同金额、开始/结束日期
- 关联客户，支持里程碑拆分付款
- 合同文件附件上传（预留）

### 3. 收款记录（含开票状态）
- 记录每笔实收金额、收款日期、付款方式
- 发票状态流转：`draft → sent → paid / overdue`
- 支持批量标记付款状态
- 逾期自动提醒（邮件/站内通知）

### 4. 收入统计图表
- 月度/季度/年度收入趋势折线图
- 按客户分布饼图
- 收款 vs 待收对比柱状图
- 数据导出（CSV / Excel）

### 5. 税务预估计算器
- 根据年度累计收入、已知扣除项，估算个税 / 增值税
- 支持自定义税率模板（适配不同省份 / 注册类型）
- 季度预缴提醒

## 四、平台与技术约束

| 项目 | 决策 |
|------|------|
| 平台 | Web（桌面优先，响应式兼容平板） |
| 金额存储 | 统一用**分（整数）**存储，避免浮点精度问题 |
| 发票状态枚举 | `draft` / `sent` / `paid` / `overdue` |
| 客户联系人 | 一客户可关联多个联系人 |

## 七、竞品对比

| 产品 | 优势 | 劣势（对自由职业者） |
|------|------|------|
| 金蝶云 | 功能全面 | 太重，学习成本高，针对企业 |
