## 约束说明

| 标记 | 说明 |
|------|------|
| 🔴 | 客户、发票标题、开票日期、到期日期、至少一行明细为必填 |
| 🔴 | 发票编号自动生成（格式 INV-YYYY-XXX），不可手动修改 |
| 🔴 | 到期日期不能早于开票日期，实时校验 |
| 🔴 | 明细行单价不能为负数 |
| 🔵 | 默认到期日期 = 开票日期 + 30 天 |
| 🔵 | 明细行小计 = 数量 × 单价，自动计算 |
| 🔵 | 明细行最多 20 行 |
| 🔵 | 税额手动填写（不同客户税率不同），单位元，自动转换为分存储 |
| 🟢 | 客户下拉支持搜索和快速创建 |
| 🟢 | 右侧实时预览发票样式 |
| 🟢 | 表单数据本地缓存（防止误关闭丢失） |
