## 约束说明

| 标记 | 位置 | 说明 |
|------|------|------|
| 🔴 | 顶部导航 | 用户未登录时强制跳转登录页 |
| 🔴 | 左侧导航 | 订阅到期后只读，创建类操作禁用并提示续费 |
| 🔵 | 统计卡片 | 年度同比数据在第一年使用时不显示（无历史数据） |
| 🔵 | 折线图 | 当月数据用虚线/不同颜色区分（未完成的月） |
| 🔵 | 逾期金额 | 为 0 时卡片用灰色，>0 时用红色主题 |
| 🟢 | 月度趋势 | 可切换为柱状图/折线图 |
| 🟢 | 按客户分布 | 点击饼图区域可跳转到对应客户详情 |
