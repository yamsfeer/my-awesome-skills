## 操作按钮区（不同状态）

### 草稿（draft）
```
[编辑] [删除]                              [→ 标记为已发送]
```

### 已发送（sent）
```
[下载 PDF]                                 [✓ 标记已收款]
```

### 已收款（paid）
```
[下载 PDF]
收款日期：2024-03-20  ✅
```
🟢 已收款后不能再改状态（只读显示收款日期）

### 逾期（overdue）
```
[下载 PDF]                                 [✓ 标记已收款]
⚠️ 此发票已逾期 5 天
```
🔴 逾期提醒必须显著展示

## 约束说明

| 标记 | 说明 |
|------|------|
| 🔴 | 只有 draft 状态可以删除，其他状态删除按钮不显示 |
| 🔴 | 只有 draft 状态可以编辑全部字段 |
| 🔴 | 总计金额必须大字粗体显示，页面视觉焦点 |
| 🔴 | 标记收款必须填写到账日期 |
| 🔵 | PDF 下载按钮仅在 sent/paid/overdue 状态显示 |
| 🔵 | 逾期发票在页面顶部用橙/红色横幅提示逾期天数 |
| 🟢 | 客户名和项目名可点击跳转对应详情页 |
| 🟢 | 发票编号支持一键复制 |
