## 约束说明

| 标记 | 说明 |
|------|------|
| 🔴 | 免责声明必须始终可见，不可隐藏 |
| 🔴 | 税务预估仅基于 paid（已收款）状态发票 |
| 🔵 | 参数修改后实时重算（防抖 500ms） |
| 🔵 | 增值税税率默认 3%（小规模纳税人），可修改 |
| 🔵 | 个人所得税采用超额累进税率速算 |
| 🔵 | 年度切换后重新加载数据 |
| 🟢 | 导出 PDF 功能（v1.5，当前灰色禁用，显示功能规划提示） |
| 🟢 | 明细计算步骤默认展开，方便用户理解计算逻辑 |
