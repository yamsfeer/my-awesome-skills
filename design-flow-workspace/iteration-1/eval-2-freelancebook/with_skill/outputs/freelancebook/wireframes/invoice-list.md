## 行内快捷操作（hover 时显示）

```
INV-2024-023   科技公司B  ¥3,200.00  [已发送📤]  2024-03-25   [标记收款] [查看] [⋮]
```

🔴 "标记收款"操作需要二次确认弹窗（选择到账日期）

## 空状态（筛选无结果）

```
┌─────────────────────────────────────────┐
│                                         │
│   筛选条件下没有找到发票                  │
│   [清除筛选]                             │
│                                         │
└─────────────────────────────────────────┘
```

🟢 筛选无结果时不显示"新建"按钮，避免干扰

## 约束说明

| 标记 | 说明 |
|------|------|
| 🔴 | 必须登录才能访问 |
| 🔴 | 逾期发票行必须用视觉区分（浅红背景或红色状态标签） |
| 🔴 | "标记收款"前必须确认到账日期 |
| 🔵 | 默认按发票创建时间倒序排列 |
| 🔵 | 每页最多显示 20 条，分页控件在底部 |
| 🔵 | 统计摘要数字随筛选条件实时更新 |
| 🟢 | 点击发票编号列跳转详情页 |
| 🟢 | 支持多选发票批量标记状态（v1.5 功能） |
