# FreelanceBook - 产品路线图

## MVP 版本（v1.0）
核心功能，解决最基本记账与开票需求：
- 用户注册/登录（邮箱 + 密码）
- 客户管理（新增/编辑/删除客户，支持多联系人）
- 项目/合同记录（绑定客户，记录合同金额、开始/结束日期）
- 收款记录（记录每笔款项，含开票状态：draft/sent/paid/overdue）
- 基础收入统计（月度收入汇总表，未到账金额）
- 订阅付费流程（30 天免费试用 → ¥39/月）

## 成长版本（v1.5）
增强体验，提升留存：
- 收入趋势折线图（月度/年度对比）
- 按客户/项目维度的收入分布饼图
- 税务预估计算器（个税速算 + 增值税估算）
- 发票到期提醒（overdue 自动提醒，邮件通知）
- 发票 PDF 导出（含公司信息、金额、税号等）
- 数据导出（CSV/Excel，供报税或第三方工具使用）
- 客户标签与筛选
- 深色模式

## 完整版本（v2.0）
商业化与平台化：
- 团队协作（支持助理/财务角色访问账本）
- 银行流水导入（CSV 自动匹配收款记录）
- 多货币支持（USD/EUR/HKD，自动汇率换算）
- Zapier/Make 集成（收款后自动触发工作流）
- API 开放（允许第三方 ERP/财税软件接入）
- 年度税务报告一键生成（适配中国个体工商户/自由职业者申报格式）
