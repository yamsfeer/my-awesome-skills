# FitStreak（健身打卡）- 产品概述

## 产品定位

FitStreak 是一款面向18-35岁上班族的健身打卡 App，通过社交激励、成就系统和数据可视化，帮助用户养成持续运动的习惯。

## 核心问题

健身爱好者难以坚持运动计划，缺乏社交激励和进度可视化。具体表现为：
- 独自运动缺乏外部监督，容易中断计划
- 进度不可见导致缺乏成就感和动力
- 无法和朋友相互激励，单打独斗效果差

## 目标用户

**主要用户群**：18-35岁城市上班族
- 有健身意愿但自律性不足
- 每天有 30-60 分钟可用于运动
- 习惯使用移动端 App，对社交功能接受度高
- 追求效率，希望用最少时间获得明显成果

## 平台

iOS + Android（React Native 跨端开发）

## 商业模式

**Freemium（免费增值）**：
- **免费版**：每日打卡、基础运动计划、好友系统（最多5位）、基础成就徽章
- **高级版（¥18/月）**：无限好友挑战、高级数据分析、AI 运动计划推荐、专属徽章、体脂趋势图、历史数据导出

## 核心功能模块

1. **每日打卡记录**：一键打卡，支持添加运动类型、时长、照片和心情
2. **运动计划制定**：按周设定运动目标，支持预设模板和自定义计划
3. **好友挑战 / 排行榜**：好友间互相挑战，周/月排行榜激励竞争
4. **成就系统 / 徽章**：完成里程碑解锁徽章，支持分享到社交平台
5. **体重 / 体脂趋势图**：记录身体数据，可视化展示健康趋势

## 核心价值主张

"把健身变成一件有趣的社交事件——每天打卡，每步可见，朋友见证你的蜕变。"
