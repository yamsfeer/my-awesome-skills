# 拾物 - 产品路线图

## MVP 版本（v1.0）
核心功能，解决最基本需求：
- 微信授权登录 + 绑定学校/校区信息
- 发布商品（最多 6 张图片 + 文字描述 + 定价 + 分类）
- 商品列表浏览（按校区过滤 + 分类筛选）
- 商品详情页（图片轮播 + 卖家信息 + 联系按钮）
- 站内私信（文字消息）
- 个人主页（我发布的 / 我想要的）
- 商品下架 / 标记已售出

## 成长版本（v1.5）
增强体验，提升留存：
- 关键词搜索（商品标题 + 描述全文搜索）
- 预约面交功能（在私信中选择时间+地点模板，生成面交卡片）
- 诚信评分系统（交易后双方互评，1-5 星 + 文字评价）
- 商品收藏 / 想要清单
- 学生证实名认证（提升平台信任度）
- 微信订阅消息通知（私信提醒、商品降价提醒）
- 推荐算法（同校区 + 浏览历史）

## 完整版本（v2.0）
商业化与平台化：
- 担保交易（引入第三方支付，2% 服务费）
- 商品竞价功能（设置最低价，买家出价）
- 校园活动闲置集市（组织线下摆摊活动的线上预报名）
- 多校区联盟（允许跨校交易，但标注距离）
- 数据分析看板（卖家查看商品曝光量、点击率）
- 违规举报与自动审核系统
