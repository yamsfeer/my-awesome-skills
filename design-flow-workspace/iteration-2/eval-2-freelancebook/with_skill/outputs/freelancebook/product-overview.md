# FreelanceBook - 产品概述

## 产品定位

面向自由职业者的一站式发票与收入管理工具——让你专注创作，告别记账烦恼。

## 核心问题

自由职业者收入来源零散（同时服务多个客户），手动记账容易遗漏，年底税务申报无从下手。FreelanceBook 把客户管理、合同记录、收款跟踪、收入统计和税务预估全部整合到一个简洁的 Web 工具中。

## 目标用户

- **群体**：独立开发者、UI/UX 设计师、咨询顾问
- **年龄**：25–45 岁
- **月收入**：2–5 万元
- **典型场景**：同时服务 3–8 个客户，混合项目制与月付合同，每季度/年底需要整理税务材料

## 平台

Web（桌面优先），响应式支持平板访问，暂不开发原生移动端

## 商业模式

订阅制 ¥39/月，提供 **30 天免费试用**（无需信用卡）。
付费功能壁垒：无限客户数、无限发票导出、税务预估报告下载

## 核心功能模块

1. **客户管理**：维护客户档案，支持每个客户多联系人，记录合作历史
2. **项目 / 合同记录**：绑定客户的项目条目，含合同金额、开始/结束日期、合同文件附件
3. **收款记录（含开票状态）**：记录每笔付款，跟踪发票状态（draft / sent / paid / overdue）
4. **收入统计图表**：按月/季/年维度可视化收入趋势，按客户/项目拆分构成
5. **税务预估计算器**：基于年度收入自动估算个人所得税应缴金额（含专项附加扣除输入）
