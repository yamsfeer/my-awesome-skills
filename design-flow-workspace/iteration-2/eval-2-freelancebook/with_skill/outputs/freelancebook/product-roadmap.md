# FreelanceBook - 产品路线图

## MVP 版本（v1.0）
核心功能，解决最基本的记账与收款跟踪需求：

- 用户注册 / 登录（邮箱 + 密码）
- 客户管理：创建 / 编辑 / 归档客户，支持多联系人
- 项目 / 合同记录：与客户绑定，记录合同金额和周期
- 收款记录：手动录入付款，关联项目，更新发票状态（draft→sent→paid/overdue）
- 收入统计图表：月度柱状图 + 客户饼图（基础版）
- 税务预估计算器：基于年累计收入的简易税率计算
- 30 天免费试用 + ¥39/月订阅付费解锁导出功能

## 成长版本（v1.5）
增强体验，提升留存与使用频率：

- 发票 PDF 生成与导出（带品牌 Logo）
- 逾期发票自动提醒（邮件 / 站内通知）
- 收入目标设定与完成进度看板
- 批量导入历史收款记录（CSV 导入）
- 客户门户：客户可通过专属链接查看并确认发票
- 专项附加扣除详细录入（子女教育、住房贷款等）

## 完整版本（v2.0）
商业化与平台化，拓展变现边界：

- 多币种支持（美元、港币、欧元）
- 年度财税报告一键导出（PDF，适配个税申报辅助）
- 开放 API / Zapier 集成（对接钉钉、飞书、微信等）
- 团队协作版（2–5 人工作室共享账本）
- 智能记账建议（AI 分类收款类型、识别异常）
- 移动端 PWA（iOS / Android 自适应）
