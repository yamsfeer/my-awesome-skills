# 足迹 - 产品路线图

## MVP 版本（v1.0）
核心功能，验证"视频分析 → 精准选品"核心链路：

### 用户模块
- 手机号注册 / 登录
- 基础个人信息填写（性别、年龄、体重）
- 修改密码 / 退出登录

### 电商模块
- 鞋垫商品列表（首页推荐 + 分类浏览）
- 商品详情页
- 购物车（增删改）
- 下单 + 微信支付 / 支付宝支付
- 收货地址管理（增删改默认地址）
- 订单列表（基础状态：待付款、待发货、已完成）

### 视频分析模块
- 拍摄引导页（分步骤提示用户如何拍摄）
- 视频录制 + 本地预览
- 视频上传
- 基础分析报告展示（足弓类型 + 风险提示 + 推荐鞋垫）

---

## 成长版本（v1.5）
增强分析精度，提升电商转化：

- 第三方登录（微信 OAuth）
- 足部档案（历史测量数据对比趋势）
- 商品评价 + 晒单
- 智能推荐首页（基于历史分析结果个性化推荐）
- 订单物流跟踪
- 优惠券 / 积分系统
- 分析报告 PDF 导出 / 分享
- Push 通知（订单状态、报告完成）

---

## 完整版本（v2.0）
商业化深化与生态构建：

- 品牌旗舰店入驻
- 直播带货模块
- 专业足科医生在线咨询
- 企业/机构版（足部健康批量筛查）
- 多语言国际化（出海东南亚市场）
- AI 模型持续迭代（更精准的步态分析）
