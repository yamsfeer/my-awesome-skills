## 页面说明

```

│       AI 足部健康管家            │
│                                 │
│                                 │
├─────────────────────────────────┤
│                                 │
│  手机号                         │
│  ┌───────────────────────────┐  │

│  └───────────────────────────┘  │
│                                 │
│  验证码                         │
```

## 交互约束
- 🔴 手机号未填写时，"发送验证码"按钮置灰不可点
- 🔴 验证码输入框在点击发送后才激活（之前为禁用态）
- 🔴 登录按钮需手机号 + 验证码同时填写才激活
- 🔵 验证码输入6位后自动提交（无需点击按钮）
- 🟢 键盘弹起时，logo 区域收起，保证输入区域可见
