# footX - 产品概述

## 产品定位

面向海外用户的 AI 智能定制鞋垫 APP——通过拍摄脚部视频，AI 分析足弓类型、步态与体态数据，为用户推荐并定制专属矫形鞋垫。

## 核心问题

大多数人不知道自己是否需要矫形支撑，也无法在不去诊所的情况下获取专业足部分析。footX 让"足部体检"触手可及，并直接连接到定制产品购买。

## 目标用户

- 海外（欧美、澳洲等）注重健康与运动表现的消费者
- 有足部疼痛、扁平足、高足弓等困扰的人群
- 运动爱好者、久站工作者（护士、厨师、零售员等）

## 平台

iOS / Android（原生移动端）

## 商业模式

按订单付费：用户完成 AI 分析后选择推荐鞋垫型号下单，每笔订单收费。分析本身免费（降低使用门槛，提升转化）。

## 核心功能模块

### 模块一：用户管理
- Apple Sign-In（iOS 必须支持）
- Google Sign-In（Android + 海外主流）
- 邮箱注册/登录（兜底方案）
- 收货地址管理（支持海外地址格式）
- 个人档案（鞋号、性别、年龄、运动偏好）

### 模块二：AI 足部分析
- 引导用户选择鞋类类型（运动鞋/正装鞋/休闲鞋等）
- 拍摄脚部视频（步行片段，前后左右多角度）
- AI 分析处理（云端）
- 分析报告展示：
  - 足弓类型（高足弓 / 正常 / 扁平足）
  - 步态分析（内旋 / 外旋 / 正常）
  - 压力分布热力图
  - 体态数据（骨盆倾斜、是否头前倾等）
  - 推荐鞋垫型号 / SKU

### 模块三：下单与支付
- 查看推荐鞋垫详情
- 加入购物车 / 直接下单
- 海外支付（Stripe / Apple Pay / PayPal）
- 订单管理（历史订单、物流追踪）
- 收货地址填写与管理

## AI 分析数据输出字段

| 字段 | 说明 |
|------|------|
| arch_type | 足弓类型：high / normal / flat |
| gait_pattern | 步态：neutral / overpronation / supination |
| pressure_map_url | 足底压力热力图 URL |
| posture_score | 体态评分（0-100） |
| forward_head | 是否头前倾：true / false |
| pelvis_tilt | 骨盆倾斜角度（度） |
| recommended_sku | 推荐鞋垫 SKU 列表（按匹配度排序） |
